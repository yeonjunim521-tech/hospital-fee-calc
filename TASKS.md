- 작업 목록 파일은 `TASKS.md` 하나만 사용한다.
- `TASKS.md`에는 현재 작업 하나만 남긴다.
- 이전 작업 기록은 `docs/TASK_HISTORY.md`에 보관한다.
- 상태 표기: `[ ]` 미완료, `[/]` 진행 중, `[x]` 완료

목표: 검사/시술/수술은 최초 미선택 상태로 두고, 대분류/중분류/소분류 선택 범위 안에서만 검색되게 한다. 별도 통합검색을 추가해 기타 처치·주사·마취까지 전체 검색하고, 기타 개별 검색 UI는 제거한다. 수술 선택 시 마취/주사 등 부대비용은 자동 산정한다.

## 진행 메모

- 현재 git 상태는 `main...origin/main` 깨끗함.
- `script.js`에는 안티그래비티 작업으로 검색 관련 함수가 중복 정의되어 있어, 실제 브라우저 동작은 하단 마지막 함수 정의를 기준으로 수정해야 함.
- 전체 통합검색은 모든 검사·시술·수술·기타 처치 항목을 검색하고, 범위검색은 선택한 대분류/중분류/소분류 안에서만 검색하도록 override 적용.
