# 작업 목록

## 운영 규칙

- 작업 목록 파일은 `TASKS.md` 하나만 사용한다.
- `TASKS.md`에는 현재 작업 하나만 남긴다.
- 이전 작업 기록은 `docs/TASK_HISTORY.md`에 보관한다.
- 상태 표기: `[ ]` 미완료, `[/]` 진행 중, `[x]` 완료

## 현재 작업: 직접선택 제목 삭제 및 수술 중분류 목록 표시 수정

목표: `직접 선택 (중분류 - 소분류)` 문구를 화면에서 제거하고, `수술 > 신경외과 수술` 선택 시 항목 목록이 즉시 나오게 고친다.

## 작업 범위

- [x] HTML(화면 구조)과 JS(동작 코드) 위치 확인
- [x] 직접선택 제목 문구 삭제
- [x] 선택된 탭 기준을 실제 활성 탭으로 고정
- [x] 수술 중분류 선택 시 목록 표시 보정
- [x] 문법/test(테스트) 검증
- [/] commit(커밋), push(푸시)

## 진행 메모

- 원인 후보: 마지막 JS override가 `activeTab` 값을 먼저 사용해 실제 선택된 탭과 어긋날 수 있음.
- 조치: 실제 활성 탭 버튼 기준으로 대분류를 읽고, 중분류 선택 시 목록 영역을 강제로 표시한다.
- 검증: `수술 > 신경외과 수술` 매칭 항목 430개 확인.
- 검증: `node --check frontend\assets\js\script.js`, `node tests\test_runner.js` 통과.
