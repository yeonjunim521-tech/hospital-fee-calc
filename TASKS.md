# 작업 목록

## 운영 규칙
- 작업 목록 파일은 `TASKS.md` 하나만 사용한다.
- `TASKS.md`에는 현재 작업 하나만 남긴다.
- 이전 작업 기록은 `docs/TASK_HISTORY.md`에 보관한다.
- 상태 표기: `[ ]` 미완료, `[/]` 진행 중, `[x]` 완료

## 현재 작업: 관리자 검색 통계 대시보드 시안 및 기능 추가

목표: 관리자 페이지에서 방문자 검색 통계를 기간별로 확인하고, 검색 결과가 없어 새로 추가한 항목도 확인할 수 있는 시안용 HTML과 API 기능을 추가한다.

## 작업 범위
- [x] 기간 필터 추가: 오늘, 24시간, 3일, 7일, 30일, 90일
- [x] 시안 2번 기준 관리자 대시보드 HTML 생성
- [x] 무결과 후 추가한 항목 저장/조회 기능 추가
- [x] build/test(빌드/테스트) 또는 가능한 검증 수행

## 진행 메모
- 서브에이전트 `Cicero`: API/DB 담당
- 서브에이전트 `Averroes`: 시안 HTML 담당
- 메인 세션: 진행표, 통합, 검수 담당
- 검증 완료: `node tests/test_runner.js`, HTML 내장 스크립트 `node --check`, `git diff --check`
- 검증 못함: TypeScript 타입 체크는 로컬 `tsc` 부재로 미실행
