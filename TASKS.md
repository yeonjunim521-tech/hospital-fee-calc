# 작업 목록

## 운영 규칙

- 작업 목록 파일은 `TASKS.md` 하나만 사용한다.
- `TASKS.md`에는 현재 작업 하나만 남긴다.
- 이전 작업 기록은 `docs/TASK_HISTORY.md`에 보관한다.
- 상태 표기: `[ ]` 미완료, `[/]` 진행 중, `[x]` 완료

## 현재 작업: 치료 및 검사 항목 분류 선택 글자색 수정

목표: `치료 및 검사 항목 추가` 영역의 중분류/소분류 선택 드롭다운에서 글자가 배경과 겹쳐 안 보이는 문제를 고친다.

## 작업 범위

- [x] 관련 HTML/CSS 위치 확인
- [x] 드롭다운 select/option 글자색과 배경색 고정
- [x] 변경 범위 확인
- [/] commit(커밋), push(푸시)

## 진행 메모

- 원인 후보: 브라우저 기본 option(선택지) 렌더링이 어두운 배경/어두운 글자 조합으로 표시됨.
- 조치: `.form-select`와 내부 `option`의 배경/글자색을 명시했다.
- 검증: 변경 파일 2개 확인, 민감정보 패턴 없음.
