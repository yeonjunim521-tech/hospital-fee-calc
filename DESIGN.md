# 관리자 검색 통계 대시보드 디자인 시스템

## 1. 제품 성격
- 대상: 내부 운영자
- 목적: 방문자 검색 로그 확인, 무결과 키워드 분류, DB 보강 후보 정리
- 인상: 차분함, 빠른 판독성, 표 중심 운영 화면, 가벼운 의료/행정 톤

## 2. 색상 토큰
- `--bg-page: #f6f8fc`
- `--bg-panel: #ffffff`
- `--bg-soft: #f2f5fa`
- `--line-soft: #e6ebf2`
- `--line-strong: #d7deea`
- `--text-strong: #1c2740`
- `--text-base: #344256`
- `--text-muted: #7b8798`
- `--blue-strong: #2f6fed`
- `--blue-soft: #eaf1ff`
- `--green-strong: #22a96f`
- `--green-soft: #e8f8f0`
- `--orange-strong: #ff8a1f`
- `--orange-soft: #fff4e8`
- `--red-strong: #ef4444`
- `--red-soft: #feecec`
- `--shadow-panel: 0 10px 30px rgba(28, 39, 64, 0.08)`

## 3. 타이포그래피
- 기본 글꼴: `"Segoe UI", "Malgun Gothic", "Apple SD Gothic Neo", sans-serif`
- 페이지 제목: `40px / 700 / -0.04em`
- 섹션 제목: `18px / 700 / -0.03em`
- 카드 수치: `26px / 700 / -0.04em`
- 본문: `14px / 500 / 1.6`
- 테이블 헤더: `12px / 700`
- 상태 배지: `12px / 700`

## 4. 간격 규칙
- 기본 단위: `4px`
- 패널 내부 여백: `24px`
- 카드 간격: `16px`
- 입력/버튼 높이: `40px ~ 44px`
- 대시보드 외곽 최대 폭: `1480px`

## 5. 컴포넌트 패턴
- 요약 카드: 흰색 배경, 얕은 그림자, 우측 아이콘 블록, 좌측 수치
- 운영 보드: 상단 제목 줄 + 얇은 구분선 + 표
- 우측 상세 패널: 선택 항목 정보, 연관 키워드 칩, 메모 입력, 하단 액션 버튼
- 상태 배지:
  - 무결과: 주황
  - 결과 있음: 초록
  - 후보/대기: 회색
  - 보류/삭제 위험: 빨강

## 6. 인터랙션
- 기간 버튼은 segmented control(분절 버튼) 형태
- 선택된 행은 연한 파랑 배경
- hover는 배경색만 약하게 변화
- 액션 버튼은 하단 고정 느낌으로 강한 색 대비 사용

## 7. 금지
- 홈 화면과 억지 통일 금지
- 과한 글래스모피즘, 과한 그라디언트 금지
- 카드 안 카드 안 카드 구조 금지
- 설명형 마케팅 카피 금지
